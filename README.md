# LAMP Docker Application

A LAMP based Docker application for use in development.

### Containers
* **lampserver-apache**: Apache, PHP, curl, nano, composer
* **lampserver-mysql**: MySQL 8
* **lamp-phpmyadmin**: phpMyadmin

### PHP Versions
* PHP 7.4.33
* PHP 8.0.30
* PHP 8.1.27
* PHP 8.2.15
* PHP 8.3.2
